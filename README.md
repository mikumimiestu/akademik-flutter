## 🚀 Quick Start

### Prerequisites

- Flutter SDK (3.0+)
- Dart SDK (2.17+)
- IDE (VS Code/Android Studio)

### Installation

1. **Clone repository**

   ```bash
   git clone https://github.com/username/akademik_group_task.git
   cd akademik_group_task
   ```

2. **Install dependencies**

   ```bash
   flutter pub get
   ```

3. **Run aplikasi**
   ```bash
   flutter run
   ```
